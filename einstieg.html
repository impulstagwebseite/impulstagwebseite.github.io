Einstieg in Webseiten

Âuf dieser Seite wollen wir uns den grundlegenden Elementen einer Webseite widmen.

Aufbau einer Webseite

  1. Der Aufbau einer Webseite besteht aus zwei wichtigen Bereichen:

    1. Head: Im "Kopf" werden grundlegende Dinge angegeben. So bestimmt man hier zum Bespiel den Titel oder den Autor einer Webseite.
    2. Body: Im "Körper" der Webseite wird der Inhalt einer Webseite abgefüllt. Hier kommen also die Text und die Bilder rein.

  2. Der Aufbau sieht also graphisch so aus:

    <html>
    <head>
    <title>Page title</title>
    </head>
    <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
    </body>
    </html>

Was sind "Tags"?

Damit wir Informationen in einer Webseite darstellen können, werden sie in sogenannte "Tags" eingefasst. Diese Markierungen dienen dazu, Inhalt als "etwas bestimmtes" zum klassifizieren.

  • Achtung: Tags müssen geöffnet und geschlossen werden!

  • Achtung: Tags können in anderen Tags drin sein

Folgende Tags werden wir anwenden: 

     <html> Hier kommt der Inhalt des Kopfes </html> 
     <body> Hier kommt der Inhalt des Körpers </body> 
     <h1> Hier kommt die Überschrift hin </h1> 
     <p> Hier kommt der Inhalt des Paragraphs hin </p>

Wichtige Dateien

Eine Webseite besteht aus vielen verschiedenen Dateien. Diese Dateien haben meist unterschiedliche Aufgaben. Hier möchte ich kurz beschreiben welche Dateien wir in unseren Arbeiten antreffen werden.

  • HTML-Dateien: Diese Dateien haben die Dateiendung .html. Sie werden benutzt um den Inhalt einer Webseite zu erfassen. Eine spezielle html-Datei ist die index.html. Diese Datei wird normalerweis immer als Erstes angezeigt.
  • CSS-Dateien: Diese Dateien haben die Dateiendung .css. Sie werden benutzt um das Aussehen einer Webseite zu steuern. Diese Dateien werden von einer html-Datei "eingebunden".
  • Bilder-Dateien: Keine Webseite ohne Bilder. Bild-Dateien können verschiedene Dateiendungen haben. Die häufigste Dateiendung ist jpg. Auch diese Bild-Dateien müssen von der html-Datei eingebunden werden.

Wichtige Programme

Damit Webseiten "schön" dargestellt werden, benötigen wir ein Programm das HTML versteht. Solche Programme heissen Internet-Browser. Mit diesen Programmen kann man eine Webseite öffnen - entweder ist diese Webseite (einfach eine HTML-Datei) im Internet oder auf dem lokalen Computer gespeichert.

Mögliche Internet-Browser sind:

  • Google Chrome
  • Mozialla Firefox
  • Microsoft Edge
  • Microsoft Internet Explorer
  • Opera

Starten mit einer einfachen Vorlage

Wir starten mit einer einfachen Vorlage - ihr findet sie unter diesem Link

  1. Ladet die Datei in einen Ordner auf eurem Computer (den Speicherort müsst ihr euch merken)
  2. "Entpackt" die Datei (rechte Maustaste auf der Datei -> "Alle extrahieren...")
  3. Klick nun bitte doppelt auf die index.html-Datei - sie sollte nun in einem Internet-Browser geöffnet werden
  4. Die selbe HTML-Datei könnt ihr gleichzeitig auch mit einem Texteditor (notepad) öffnen (Rechte Maustaste, Öffen -> Notepad auswählen)
