Github einrichten

Was ist Github?

Wir wollen unsere Webseite nach dem Ende des Kurs im Internet sehen können. Github ist ein Werkzeug, mit dem wir Eure Webseite einrichten, bearbeiten und anzeigen können.

Login erstellen - erste Schritte

In einem ersten Schritt geht Ihr bitte auf http://github.com und klickt auf den Button "Sign up for GitHub"



Fülle nun das Formular mit Deinen Angaben aus - Achtung: Deine Webseite wird später "BenutzerName.github.io" heissen. Wähle also einen Benutzernamen der Dir gefällt, aber nicht Deinem richtigen Namen entspricht.



Beim Passwort musst Du darauf achten, dass es genügend kompliziert ist (Mindestens 8 Zeichen, inklusive eines Zahl und gemischt mit Gross- und Kleinschreibung)

Login erstellen - zweite Schritte

Weiter unten musst Du noch ein Rätsel lösen - Dreh die Figur mit den Zeigern bis sie auf den Füssen steht und bestätige es über den Button



Bestätige abschliessend noch die Erstellung von Deinem Konto.



Bestätige den zweiten Schritt einfach über den Button.



Bestätige den dritten Schritt einfach über den Button (Du kannst etwas aus den Optionen auswählen, Du musst aber nicht).



Bestätigung der Email-Adresse und erster Login

Github hat Dir nun eine Email geschickt. Öffne bitte diese Email in Deinem Emailprogramm (oder über eine Webseite) und klicke bitte auf den Button.



Du wirst nun automatisch auf die Webseite von Github.com umgeleitet und eingeloggt.

Erstellen des Speicherorts für unsere Webseite

Damit wir später unsere Dateien (Webseite, Bilder, usw.) speichern können, müssen wir nun auf Github einige Dinge einrichten - Wir brauchen ein eigenes "Repository".

Bei der Erstellung von Deinem Repository musst jetzt auf Folgendes achten:

  • Der Repository-Name muss aus Deinem Benutzernamen, gefolgt von "github.io" bestehen.
    Beispiel: Wenn Dein Benutzername zum "ben1" wäre, müsste Dein Repository "ben1.github.io" heissen.
  • Aktiviere das Häckchen bei "Initialize this repository with a README"

Bestätige abschliessend die Erstellung über den Button ganz unten.
